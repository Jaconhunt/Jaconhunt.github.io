悬浮小球原理探究

发表于 2019-11-25 分类于 趣味物理

悬浮小球

关于小球为何能够进行悬浮，网络上给出的简单解释几乎都为伯努利原理，但对小球/乒乓球/气球的受力分析并未给出详细的解答，故在此探究小球能够悬浮的真正原因。

通常我们还会看到其他类似的现象，但解释起来并不轻松，在此也一并记录。

a、乒乓球/气球悬浮，介质时空气

b、在水龙头下放乒乓球被吸住

c、圆盘型物体悬浮，介质时向上喷射的水流

d、用漏斗吹气，乒乓球没有被吹走，而被吸住

e、一张白纸上放两个乒乓球，朝中间吹气，乒乓球靠近
